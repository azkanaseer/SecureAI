SQL Injection Detected

Direct interpolation of the dynamic variables user and password into raw SQL strings allows total authentication bypass.

SCORE: 50/100
const userInput = getQueryParam('info'); const textContainer = document.getElementById('output'); textContainer.innerHTML = userInput;

DOM XSS Vulnerability

Direct assignment of dynamic unescaped browser search queries into innerHTML enables custom JavaScript injection attacks.

Stack Buffer Overflow

Using unbounded standard terminal reads into fixed 16-byte character buffers allows stack memory allocation overrides.
