# SecureAI — AI Powered Code Security Reviewer

## 🔗 Live Demo
https://secure-ai-eight.vercel.app/

## About
A professional web application where developers or any person can paste their code and instantly get a detailed AI-powered security audit report.

## Features
- Instant vulnerability detection
- AI-powered security score (0-100)
- Suggested fixes for each issue
- 50+ vulnerability types detected
- Light/Dark mode toggle
- Share results as link
- No code stored — 100% private

## Tech Stack
- HTML, CSS, JavaScript
- Gemini AI API
- Deployed on Vercel
